📊 Credit Risk & Stress Testing Dashboard
🚀 Overview

An end-to-end machine learning project to predict loan default risk and simulate stress scenarios using an interactive Streamlit dashboard.

🎯 Features
Credit risk prediction (Low / Medium / High)
Interactive Streamlit dashboard
Stress testing (income, loan changes)
Model evaluation (ROC-AUC, confusion matrix)
🧠 Tech Stack

Python, Pandas, Scikit-learn, Streamlit

🌐 Live App

👉 https://credit-risk-and-stress-testing-with-dashboard-14.streamlit.app/
